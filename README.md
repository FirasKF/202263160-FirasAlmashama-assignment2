# Assignment 2 – Interactive Portfolio Website

## 📌 Project Description
This project is an improved version of my personal portfolio website developed in Assignment 1.  
The goal of this assignment is to enhance the website by adding interactivity, dynamic content, and better user experience using JavaScript.

The website showcases my background, skills, and projects in a modern and interactive way.

---

## 🚀 Features

### 🔹 Dynamic Content
- Projects are displayed dynamically using JavaScript
- Users can:
  - Filter projects by category
  - Search projects using a live search bar

### 🔹 Data Handling
- Theme preference (light/dark mode) is saved using `localStorage`
- Fun fact section fetches data from a public API

### 🔹 Interactivity
- Navigation menu toggle for mobile view
- Buttons and user actions dynamically update content

### 🔹 Form Validation
- Contact form validates:
  - Name (required)
  - Email (valid format)
  - Message (minimum length)
- Displays error and success messages

### 🔹 Animation & UI Enhancements
- Smooth fade-in animations on scroll
- Hover effects on cards and buttons
- Responsive design for different screen sizes

### 🔹 User Feedback
- Loading indicator for API requests
- Error message if API fails
- Empty state message when no projects match search/filter
